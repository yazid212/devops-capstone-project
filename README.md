## Project Description

This is the **devops-capstone-project** repository created from the IBM DevOps Capstone Project template. This project will implement a complete DevOps pipeline for a Flask microservice, starting with implementing CRUD operations using Test-Driven Development (TDD), achieving 95% test coverage, containerizing with Docker, and deploying through CI/CD pipelines to Kubernetes/OpenShift.

## Current Status
- ✅ Repository created from template
- ⬜ Implementing CRUD operations with TDD
- ⬜ Achieving 95% test coverage
- ⬜ Containerization with Docker
- ⬜ CI/CD pipeline implementation
- ⬜ Kubernetes/OpenShift deployment

## Next Steps
1. Complete the Account microservice with full CRUD operations
2. Write comprehensive tests following TDD principles
3. Achieve at least 95% code coverage
4. Containerize the application
5. Build and deploy through CI/CD pipeline
